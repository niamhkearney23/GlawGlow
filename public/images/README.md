# Images

Drop your photos here using these **exact file names**. Until a file exists,
the site shows an intentional cocoa→bronze placeholder with a label — so the
site never looks broken while you gather photos.

| File name        | Where it shows           | Which photo to use                          |
| ---------------- | ------------------------ | ------------------------------------------- |
| `logo.png`       | Nav logo badge           | Your Glam & Glow logo (square)              |
| `hero.jpg`       | Full-screen hero         | Spray-tan editorial body shot (moody one)   |
| `split.jpg`      | "Your artist" row        | Front-facing lash lift / brow photo         |
| `gallery-1.jpg`  | Gallery — tile 1         | Brow wax                                     |
| `gallery-2.jpg`  | Gallery — tile 2         | Brow shape, wax & tint                       |
| `gallery-3.jpg`  | Gallery — tile 3         | Lash lift, tint & brow lamination            |
| `gallery-4.jpg`  | Gallery — tile 4         | Spray tan                                    |
| `gallery-5.jpg`  | Gallery — tile 5         | Spray tan (second)                           |

Note: a couple of photos are used in two places (e.g. the lash/brow front shot
works as both `split.jpg` and `gallery-3.jpg`). Just upload the same file under
both names.

## How to add a photo (easiest way — on GitHub)

1. Open this folder on GitHub: `public/images`
2. Click **Add file → Upload files**
3. Make sure each photo is named exactly as above (e.g. `hero.jpg`, all lowercase)
4. Click **Commit changes** — Vercel redeploys automatically in ~1 minute

To change a caption or which file a section points to, edit `src/config/content.ts`.
