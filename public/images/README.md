# Images

Drop your photos here using these **exact file names**. Until a file exists,
the site shows an intentional cocoa→bronze placeholder with a label — so the
site never looks broken while you gather photos.

| File name        | Where it shows         | Suggested size      |
| ---------------- | ---------------------- | ------------------- |
| `logo.png`       | Nav logo badge         | 200 × 200 (square)  |
| `hero.jpg`       | Full-screen hero       | 2000 × 1200 (wide)  |
| `split.jpg`      | "Your artist" row      | 1200 × 1500 (tall)  |
| `gallery-1.jpg`  | Gallery tile 1         | 1000 × 1000 (square)|
| `gallery-2.jpg`  | Gallery tile 2         | 1000 × 1000 (square)|
| `gallery-3.jpg`  | Gallery tile 3         | 1000 × 1000 (square)|
| `gallery-4.jpg`  | Gallery tile 4         | 1000 × 1000 (square)|
| `gallery-5.jpg`  | Gallery tile 5         | 1000 × 1000 (square)|
| `gallery-6.jpg`  | Gallery tile 6         | 1000 × 1000 (square)|

## How to add a photo (easiest way — on GitHub)

1. Open this folder on GitHub: `public/images`
2. Click **Add file → Upload files**
3. Make sure each photo is named exactly as above (e.g. `hero.jpg`)
4. Click **Commit changes** — Vercel redeploys automatically in ~1 minute

To change which file a section points to, edit `src/config/content.ts`.
